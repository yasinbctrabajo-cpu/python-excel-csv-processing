# Procesamiento de Excel y CSV con Python

## Descripción

Repositorio que contiene ejercicios prácticos relacionados con la lectura, escritura y manipulación de archivos Excel y CSV utilizando Python.

## Tecnologías utilizadas

- Python 3
- CSV
- Excel
- Jupyter Notebook

## Conceptos trabajados

- Lectura de archivos CSV
- Escritura de archivos CSV
- Procesamiento de datos
- Manipulación de información
- Automatización de tareas
- Gestión de archivos

## Objetivo

Aprender a trabajar con datos almacenados en archivos Excel y CSV para automatizar procesos y gestionar información de forma eficiente.
